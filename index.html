How the Website Works

This website is designed for anime fans who want personalized recommendations based on their MyAnimeList (MAL) watch history. Here’s how it works:

1. Uploading the MAL Export File

  • First, export your anime list from MyAnimeList in XML format.

  • Then, upload this XML file to the website using the Browse button.

2. Processing the File

  • When you click "Convert", the website reads and parses the XML file.

  • It filters out only completed anime from your list.

  • It extracts the titles and ratings of these completed anime.

  • If multiple entries share a similar base title (e.g., Attack on Titan and Attack on Titan: Final Season), they are grouped together, and their average rating is calculated.

3. Generating a Markdown-Based AI Prompt

  • The processed data is formatted into a Markdown prompt.

  • The prompt describes your anime preferences and includes a table of watched anime with ratings.

  • The AI is instructed to recommend 10 new anime based on genres and themes from your highly-rated shows.

4. Copying the Prompt

  • Click "Copy Prompt" to copy the generated Markdown text.

  • Then, paste it into a Large Language Model (LLM) (like ChatGPT) to receive anime recommendations.

Key Features

✅ Simple Interface – Just upload, convert, and copy.

✅ AI-Optimized Prompt – Ensures better recommendations.

✅ Title Grouping – Merges similar titles for cleaner data.

✅ Excludes Sequels/Prequels – Focuses on fresh recommendations.

In short, this tool automates the process of analyzing your MAL history and formatting it into an AI-friendly structure, making it easy to get highly personalized anime recommendations.
